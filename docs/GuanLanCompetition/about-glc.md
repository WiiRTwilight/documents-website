# 关于观澜杯2026

观澜杯是由深圳市观澜中学2026届 (按照高考时间算) 某班学生组织的一场赛事，以《我的世界》作为赛事选中的游戏。第一届比赛将在2026年_月_日进行。

在比赛前3-5天，比赛联机房间将会提前开放，以使得玩家提前适应和训练技能。

## 比赛内容

### 起床战争

起床战争是《我的世界》各多人服务器中相当热门的竞技性小游戏。你需要通过资源生成点收集物资，在商店兑换物品，提防他人破坏你家的床并破坏他人的床，直至场上还剩最后一个队伍为止。

#### 赛制

- 8=<人数=<16时，所有人进行同一场比赛。
- 人数>16时，抽签分场次进行比赛。

### 警匪大战

警匪大战是知名第一人称射击游戏《反恐精英》在《我的世界》上的复刻，以快节奏，高竞技性，严格的经济分配策略要求以及较高的还原度风靡于多个《我的世界》多人服务器。

#### 赛制

- 4=<人数=<10时，所有人进行同场比赛。
- 人数>10时，抽签分场次进行比赛。
- **人数<4时，不进行比赛。**

### 极限生存冠军

极限生存冠军基于《我的世界》的原版生存玩法，对玩家的生存能力，血量管理和战斗能力有较高要求。玩家需要在前10分钟的非PVP时间相互远离，中间50分钟的时间尽可能发展出好的装备和工具，准备最后的对决。

#### 赛制

所有玩家进行同一场比赛。

## 比赛人数，规则以及奖品

在每一局游戏获胜的队伍及其玩家，手机玩家记4分，电脑玩家记三分。

如果自动反作弊与人工核对流程中发现作弊现象，如非正常组队，使用作弊软件取得不当优势等，对局结果取消，清除作弊玩家后另开一局。

当玩家人数未达6人，不设奖品。

当玩家数量达到6人，冠军获得125元。

当玩家数量达到12人，冠亚军分别获得125元和75元。

当玩家数量达到24人，冠亚季军分别获得125元，100元和75元。

如果想要《我的世界》正版账号，也可换成一个账号以及剩余奖金 (比如，100元对应一个正版账号和11元).
