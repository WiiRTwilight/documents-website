# 关于 YSM

Yes Steve Model 模组可以用于修改《我的世界》的玩家模型，类似于此前发展时间较久的[Figura](https://www.mcmod.cn/class/3709.html)和远古玩家记忆中的[自定义史蒂夫](https://www.mcmod.cn/class/672.html)模组。其对游戏玩法本身没有很大的影响，但因其可以更改玩家外观，且社区存在大量制作精湛的方块化的人物模型，使得该模组在国内的讨论度较高；同时，这种方块化的人物模型明显比原版更精致，也避免了高精度模型的同质化和学习成本高的问题，这类模型也广受玩家社区的喜爱，使得YSM模型社区逐渐形成了标志性的模型风格。(这种模型的风格标志性到你大部分时候甚至可以一眼看出这是给YSM模组做的模型，也算是YSM模组的标志之一了)

## YSM 的优势

如上文所述(并附有补充)，YSM的优势源于其模型社区的艺术劳动结晶和玩家对虚拟自我的身份认同需要，来自模组本身或者其他因素的优势并不占主导因素。
