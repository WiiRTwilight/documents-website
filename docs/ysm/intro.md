# Twilight 的 Yes Steve Model 文档

众所周知，Yes Steve Model 模组 (下文简称YSM) 在现今的国内我的世界圈子的讨论度很高。其修改玩家模型的功能，以及大量比原版史蒂夫更精致的模型的存在，使得该模组的热度在国内范围持续走高。

该部分文档并不会专注于 YSM 本体的安装，功能和其他方面，相反，暮光更倾向于在这里聊一些闲话，以及介绍一些很实用的第三方YSM附属软件。
