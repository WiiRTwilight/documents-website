## 关于 FreeBSD

[FreeBSD](https://freebsd.org) 及其所属的整个 BSD 软件系列，在千禧年之后都是一个出场率极高，存在感极低的软件。其常被简单认为是一种 Unix (抑或是其自古以来不可分割的一部分)，或是苹果公司所开发 Darwin 操作系统系列的基础。而 FreeBSD 则是 BSD 这一整个大家族中少数在互联网还有其身影，并还能留下一些深刻印象的寒门子弟中的佼佼者。

::: info
BSD 的没落并不是因为它菜，正如古代寒门子弟之所以是寒门并不是因为贫穷一样。
:::
